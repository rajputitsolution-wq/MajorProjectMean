## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/project-name.git
cd project-name
```

### 2. Install Backend Dependencies

```bash
cd Back-End
npm install
```
### 3. Install Frontend Dependencies

```bash
cd Front-End
npm install
```


### Start Backend

```bash
cd Back-End
npm start
```

Backend runs on:

```
http://localhost:5000
```
### Start Angular Frontend

```bash
cd Front-End
ng serve
```

Frontend runs on:

```
http://localhost:4200
```
